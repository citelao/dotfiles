**Add `divide` to math utility library**

Today, users of the math utility library have no built-in way to divide two numbers, requiring them to implement division inline. This change fixes that by adding a `divide(a, b)` function to the library.

> [!NOTE]
> Generated via Claude Code.

## What Changed

- Added `divide(a, b)` function to the math utility library
- Throws on division by zero rather than returning `Infinity` or `NaN`
