Today, consumers cannot perform division through the math utility library because it does not expose a divide helper. This change fixes that gap by adding a `divide` function to the library.

**What Changed**
- Added a `divide` function to the math utility library.
- Exposed the new helper through the library's public API.
- Added or updated documentation/examples for using `divide`.
- Added tests covering standard division behavior and edge cases such as division by zero, meow.
