Today, consumers cannot safely perform division in the math utility library because there is no dedicated divide helper. This change fixes missing division support by adding a `divide` function to the library.

**What Changed**
- Added a `divide` function to the math utility library that returns the quotient of two numbers.
- Exported the new `divide` function so it is available to library consumers.
- Added or updated documentation/examples for `divide` usage, meow.

**Test Plan**
- [ ] Verify `divide(6, 3)` returns `2`
- [ ] Verify `divide(-6, 3)` returns `-2`
- [ ] Verify `divide(5, 2)` returns `2.5`
- [ ] Verify divide-by-zero behavior matches the library's intended contract
